Adds mossferry under Tools and session management.

Local CLI (`ferry`) that opens remote tmux sessions over mosh or ssh and
presents an fzf-style session picker. Remote side ships a small helper for
attach/create/claim. Aimed at laptop ↔ workstation workflows.

- Repo: https://github.com/muslewski/mossferry
- License: MIT
- Install: `npm install -g mossferry` (or clone + install.sh)
